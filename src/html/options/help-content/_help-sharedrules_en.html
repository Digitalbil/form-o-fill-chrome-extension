Sharing field definitions

It is possible to share field definitions between rules.
This enables you to write a set of field definitions once and insert those somewhere in another rule's workflow.

Imaging a rule that fills a long registration form with data.
Also imaging that you need to test that form three times with the only difference being a single input field.

You could create a single rule without that input field and three rules each importing the shared field definitions and the input field with the unique value.
