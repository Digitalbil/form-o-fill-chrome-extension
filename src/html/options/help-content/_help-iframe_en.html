Selecting forms inside iframes

If the form you want fill is contained inside an <iframe> you can use the within property of a field definition to scope the selector to the frame.






var rules = [{
  name: "This is my first rule. So excited!",
  url: /^http.*test.html$/,
  fields: [{
Form-O-Fill will look for a <iframe> tag but you can use any CSS selector here.
Just make sure the within selector matches only one element.
    within: "iframe",
The selector to be used inside the iframe.
    selector: "input[type=text]",
The value to fill the field with.
    value: "Nyan Nyan Nyan",
 
  }]
 
}];
