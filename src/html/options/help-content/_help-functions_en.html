Using functions as values

Using static strings/text as values when filling out forms isn't enough.
You can use functions as values and dynamically create the value to fill the field with.

This is how it works:

Inside a field definition ...
fields: [{
Find the field ...
  selector: "input[type=text]",
Create a function. The function will receive two paramters:
$element: The jQuery wrapped element that is going to be filled.
$data: Data from before functions [optional]
  value: function($element, $data) {
Return the value the field should be filled with.
    return $element.val().toLowerCase();
 
  }]
