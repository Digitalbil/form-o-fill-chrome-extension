Using libraries

Form-O-Fill ships with two powerful libraries for creating test data.
To create random data you can use chance.js.
To create and manipulate dates, Form-O-Fill provides moment.js.

Both libraries are available as Libs.chance and Libs.moment

This is how to use them:

Inside a field definition ...
fields: [{
Find the field ...
  selector: "input[type=text]",
Use chance.js to create some data ...
  value: function($element, $data) {
.. like a random name
    return Libs.chance.name();
 
  }
 
}, {
Find a date field ...
  selector: "input[type=date]",
Use moment.js to fill it ...
  value: function($element, $data) {
.. with the current date (in you locale)
    return Libs.moment().format();
 
  }
 
}]
