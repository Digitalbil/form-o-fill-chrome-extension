Using libraries

Form-O-Fill ships with two powerful libraries for creating test data.
To create random data you can use chance.js.
To create and manipulate dates, Form-O-Fill provides moment.js and if you are really into
math then math.js is also included.

The libraries are available as Libs.chance, Libs.moment and Libs.math.

There is also a library accessible as Libs.h which contains functions to use in your value function.

This is how to use the library functions:

function Purpose
Libs.chance.[ChanceJsFunction] Calls chance.js function
Libs.moment().[MomentJsFunction] Calls moment.js function
Libs.math.[MathJsFunction] Calls math.js function
Libs.h.click Click the element
Libs.h.copyValue(".some-selector") Copy the current value of that element to this element
Libs.h.select Select a radio or checkbox field
Libs.h.unselect Unselect a radio or checkbox field
Libs.h.stripTags("<a>SomeHTML</a>") Remove HTML tags from a string
Libs.h.screenshot("filename.png") Save a screenshot
Inside a field definition ...
fields: [{
Find the field ...
  selector: "input[type=text]",
Use chance.js to create some data ...
  value: function($element, $data) {
.. like a random name
    return Libs.chance.name();
 
  }
 
}, {
Find a date field ...
  selector: "input[type=date]",
Use moment.js to fill it ...
  value: function($element, $data) {
.. with the current date (in you locale)
    return Libs.moment().format();
 
}, {
  selector: "input[type=text]",
Use math.js to fill it ...
  value: function($element, $data) {
.. with some calculated units
    return Libs.math.add(Libs.math.unit(1, "cm"), Libs.math.unit(10, "cm"));
 
}, {
Find something clickable
  selector: "a[href]",
Use Libs.h.click to click that element
Notice that there are no brackets after Libs.h.click!
  value: Libs.h.click
 
}, {
Copying values
  selector: "input[name='like-the-other-one']",
Use Libs.h.copyValue("#someselector") to copy the value of that element at the time of invocation
  value: Libs.h.copyValue("#some-field")
 
}, {
Force select/unselect state of
a input[type=radio] or input[type=checkbox].
  selector: "input[type=checkbox]",
Use Libs.h.select or Libs.h.unselect to select or unselect fields.
No brackets at the end.
  value: Libs.h.select
 
}, {
 
  selector: "input[type='text']",
Strip HTML from strings. Useful for the return value of
context.findHtml.
 
  value: function($element, $data) {
    return Libs.h.stripTags($data.someHtml);
  }
 
}]
