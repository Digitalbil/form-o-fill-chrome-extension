Filling forms

Filling forms is easy. When Form-O-Fill finds a matching rule for the site you are currently on, it will display the number of matching rules in it's button.

  • If the number of matching rules is zero, the extension will display "n/a" ().
  • If the number of matching rules is one, the extension will display "1" (). Clicking on the button will then fill the matching form with the rule's data.
  • If there are multiple matches and you click on the Form-O-Fill button you will be presented with a popup where you can select (by clicking) the appropriate rule.

In case of an error

Since you can use functions as values there may be errors executing those when filling the form.
Those errors will reported as an notification.

Clicking on the notification will bring you back to the rules editor and display the error to you.
