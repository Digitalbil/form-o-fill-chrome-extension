Basic rule syntax

Rules are written as a javascript object.
You should always start with var rules = [];.
If you do not start your rule definition with that, Form-O-Fill will complain and offer to fix that for you.

This is the structure of a basic rule:

Define an array of rules. Rules must always start with this.
var rules = [
Start of one rule
{
A regular expression that is matched against the current URL (including http:// and the like).
  url: /^http.*test.html$/,
A name for this rule. Used when multiple matches are available.
  name: "This is my first rule. So excited!",
This is an array of fields you want to fill.
  fields: [
A field hat a selector and an value.
  {
A jQuery compatible selector. (Actually document.querySelectorAll)
    selector: "#myfield",
The value to fill the field with.
    value: "Nyan Nyan Nyan"
 
  }]
 
}];
