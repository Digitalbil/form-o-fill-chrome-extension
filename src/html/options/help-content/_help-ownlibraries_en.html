Creating your own library of functions

Besides using moment.js and chance.js to create testdata you call always roll your own functions.
Have a before function duplicated in multiple rules? Abstract that away into library functions.

This is how to create your own functions:

Create a new tab and open up an array of objects:
var libs = [{
The name of the function. Later you can access this function via Libs.fName()
  export: "yourFunctionName",
The functions code:
  lib: function () {
Your code goes here...
    return "Hello from the library function 'yourFunctionName'";
 
  }
 
}];

All functions declared in this way (even those declared inside normal rules) are globally accessible.
Use them in you before- or value functions using Libs.yourFunctionName().

If you define a function with the same name multiple times it's quite unpredictable which one will get executed. So name you exports uniquely.
