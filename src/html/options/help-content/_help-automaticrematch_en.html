Automatic rematch of rules

Form-O-Fill searches for matching rules whenever you visit a webpage or the URL changes.
This is usually enough but for those cases where you need a re-matching of the rules without
one of those two events mentioned above you can use a feature called "Automatic rematch".

If enabled Form-O-Fill will search for matching rules every two seconds.

This is very useful for apps that change the DOM on the fly.
Those changes will not be detected if automatic rematch is turned off.

There are two ways to toggle automatic rematch on or off.
You can use the option in the settings panel or click "Automatic rematch" in the extension popup:

The popup text will turn red if not active and turn green if the automatic rematch is active.
